# Mahakavi Bharathiyar Tribute Webpage

## Project Overview
This project is a tribute to Mahakavi Bharathiyar, a renowned Tamil poet, freedom fighter, and social reformer. The objective of this webpage is to celebrate his life and contributions to Tamil literature and society.

## Features
- **Biography**: Detailed information about Mahakavi Bharathiyar's life.
- **Literary Works**: A section dedicated to his most influential poems and writings.
- **Legacy**: Information on the impact of Bharathiyar's work on modern Tamil culture and literature.
- **Gallery**: A visual collection of images and artifacts related to Bharathiyar.
- **Contact**: A form to reach out to the project team for any inquiries or collaborations.

## Technologies Used
- HTML
- CSS
- JavaScript

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/sakthiveljune03-hub/OIB-SIP-_Web-Development_level2_task2.git
   ```
2. Open `index.html` in your preferred web browser.

## Usage
Simply navigate through the webpage to learn about Mahakavi Bharathiyar's contributions and explore various sections to access his works and impact.
